TMDb API Key

Search is powered by The Movie Database (free). It is stored only in this browser's local storage. Don't have a key yet?

  1. Make a free account at themoviedb.org and confirm your email.
  2. Go to Settings → API → Create → Developer and fill in the short form.
  3. Copy your API Key (v3 auth) and paste it below. A v4 Read Access Token works too if you'd rather use that.

Notifications

Desktop notifications fire for your release reminders, but only while this page is open in the browser (there's no background server running).

Reminders

New titles you add default to a release-day reminder. This applies that same default to everything already in your library. Anything you've set to a specific time is left alone.

Import Watch History (CSV)

Import a watch-history CSV from Netflix (Account → Profiles → Viewing activity → Download all), Letterboxd, IMDb, Trakt, or any CSV with a title column. Headers are detected automatically (title, date, year, rating, type). Episodes are grouped per show and matched on TMDb; movies land in Watched, episode progress is set to the furthest episode found, and ratings are imported where present (5-star scales are converted to 10). You'll get a review list to approve before anything is added.

Cloud sync

Connect Google Drive to back your library up and keep it in sync across your devices. It saves to a single file in your own Drive, private to you, and the app can only ever see that one file.

Backup

Your library lives only in this browser's local storage. Export a JSON backup occasionally, or move your data to another browser.

This product uses the TMDB API but is not endorsed or certified by TMDB. All data is stored locally in your browser.
